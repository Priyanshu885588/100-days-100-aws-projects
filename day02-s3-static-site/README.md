# Day 2 — Static Website Hosting on Amazon S3

## 📌 Overview

Hosted a static website using Amazon S3 with public access.

## 🏗️ Architecture

User → S3 Static Website Endpoint → HTML Content

## ⚙️ Services Used

- Amazon S3

## 🚀 Steps

1. Create S3 bucket
2. Enable static website hosting
3. Upload HTML files
4. Set bucket policy for public read
5. Access via endpoint URL

## 🌍 Website URL

http://<bucket-name>.s3-website-<region>.amazonaws.com

## 📚 Learnings

- Object storage basics
- Static hosting
- Bucket policies
